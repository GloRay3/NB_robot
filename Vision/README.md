# 视觉与自瞄

> - 背景概要
>
你很不幸的穿越到了美剧**黑袍纠察队**。站在你面前的是**Homelander**。
![Homelander](../photos/Homelander.jpeg)
这是本剧中的最大**反派**。  
你正好撞见了他**行凶**。他已经发现了你，并且露出了他标志性的微笑，准备将你**灭口**。  
正在这危急存亡之际，你发现你的身边竟然恰好有一门**粒子激光炮**！它可以对**祖国人**造成行之有效的伤害，保住你的性命。  
又又又非常不幸的是，这门激光炮没有手动控制的接口。你必须编写简单的**自瞄**程序，来控制这门激光炮，以保住你的性命。  
祝你好运！**God Bless You!**
> - 任务配置
>
基于**ubuntu22.04 LTS**,以及**ros2 humble**。部署**yolov8**模型，实现目标检测功能。  
文件里已提供了需要进行检测的**视频源(orivedio.mp4)文件**。您可以使用官方提供的最小预训练检测模型**yolov8n.pt**文件进行模型推理。
